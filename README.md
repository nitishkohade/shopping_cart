# My Shopping Cart Application

## Prerequisites
- Java JDK 17
- Maven 3.8.1

# how to run the app in development

**Build the Project**: In the root directory of the project, run:
mvn clean install

**Run the application**:
java -jar target/shopping-cart-0.0.1-SNAPSHOT.jar
